이번 시간에는 Next.js에 CSS를 배워보자.

1.자산, 메타 데이터 및 CSS

현재 추가한 두번째 페이지는 스타일이 없다. 페이지 스타일을 지정하기 위해 css추가해보자.

Next.js에서는 CSS 및 Sass를 기본적으로 지원한다. 이 과정에서는 CSS를 사용한다.

이 강의에서는 Next.js가 이미지와 같은 정적 자산 및 `<title>`태그와 같은 페이지 메타 데이터를

처리하는 방법에 대해서도 설명한다.

이강의 에서 배울내용은 다음과 같다

1.Next.js에 정적파일 (이미지 등) 을 추가하는 법,
2. `<head>`각페이지 의 내용을 사용자 정의하는 법,
3. CSS 모듈을 사용해 스타일이 지정된 재사용 가능한 React 구성 요소를 만드는 법,
4. 어떤식으로 `pages/app.js`에 Global한 Css를 추가하는가
5. Next.js의 스타일링에 대한 몇 가지 유용한 팁.

은 다루지 않는다.
