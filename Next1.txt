
next js 앱을 시작하려면 npm run dev 후 실행

Next.js에서 페이지는 `pages`디렉토리의 파일에서 내보낸 React Component 이다.

페이지는 파일이름 에 따라 경로와 연결된다. 예를 들어 

`pages/index``/` 경로와 연결되어있다.
`pages/posts/first-post.js``/posts/first-post`경로와 연결 

이미 `pages/index.js` 파일이 있으므로 `pages/posts/first-post.js` 작동 방식을 확인 하기 

위해서 만들어 보자.
